Fractal Tree Generator

In geometry, a fractal canopy, a type of fractal tree, is one of the easiest-to-create types of fractals. Each canopy is created by splitting a line segment into two smaller segments at the end (symmetric binary tree), and then splitting the two smaller segments as well, and so on, infinitely. Canopies are distinguished by the angle between concurrent adjacent segments and ratio between lengths of successive segments.

— Wikiwand

I built this project using HTML5 Canvas and Javascript. The project generates a fractal tree based on user inputs like angle, length ratio, and depth. The tree is drawn recursively using the canvas API.

Chip 8 Emulator

CHIP-8 is an interpreted programming language, developed by Joseph Weisbecker. It was initially used on the COSMAC VIP and other computers of the mid-1970s. CHIP-8 programs are run on a CHIP-8 virtual machine.

I built this project using Typescript and HTML5 Canvas. The project emulates the CHIP-8 virtual machine, allowing users to run CHIP-8 programs on their computers. The emulator includes features like graphics rendering, input handling, and sounds.

I wanted to build this project to understand how emulators work. Whenever, someone search on the internet about the emulation building, they are recommended to build the Chip8 Emulator first to get their hands dirty.

So, I tried building this project. It is not a complicated emulation project to begin with, but there are a lots of nuances to take care of when building the emulator. Whenever one is building the chip-8 emulator, they need to make sure they test every instructions properly. Because if you don't, you will end up **staring** at your code for hours.

Raptor Interpreter

This project is also an emulator project and it is one of my favorite personal projects that I have worked on.

Little background on what this project is all about. So, the college I currently study at, The Lambton College, on my first semester, I needed to learn the basics of programming: loops, conditions, functions, etc. using a tool called Raptor.

The raptor is a flowchart-based programming environment, which allows students to visualize their algorithms. It uses flowcharts to represent the control flow of a program and allows students to experiment with different algorithms. The raptor is a tool that allows students to experiment with different algorithms and programming languages.

Whenever you save the flowchart in raptor, it saves the file with a .rap extension. So, I wanted to build a tool that can decode what the .rap files holds and interpret it using Node.js

After weeks of hard work, and a lot of digging through the files, I was able to build the tool I was hoping to build.
